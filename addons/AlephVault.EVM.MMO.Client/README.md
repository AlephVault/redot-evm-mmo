on [AlephVault.EVM.MMO.Common](../AlephVault.EVM.MMO.Common/README.md).

Read the server package documentation for the mirrored server-side setup:
[AlephVault.EVM.MMO.Server](../AlephVault.EVM.MMO.Server/README.md).
